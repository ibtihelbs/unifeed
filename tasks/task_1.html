Steps:

  1. Go to desktop:

    • open CMD
    • type : "cd desktop"

  2. clone project:

    On CMD type : "git clone https://github.com/ibtihelbs/15_min_challenge0.1"

  3. go to the generated project directory :

    on cmd type : "cd 15_min_challenge0.1"

  4. open VS code

    on cmd type : "code ."

  5. upload project to your github repository :
    1. create a new repository in you github by the name "unifeed"
    2. remove .git file in you project
    3. git init
    4. git add .
    5. git commit -m "first commit"
    6. git branch -M main(optinal)
    7. git remote add origin URL
    8. git push -u origin master (or main in case you renamed the default branch master)
    these steps are mentioned in the github repository doc
  6. Go through the project directory:
    • content file
    • style guide file
    • design and assets directories
